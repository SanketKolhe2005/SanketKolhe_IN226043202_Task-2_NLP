📌 Sentiment Analysis using NLP Pipeline & Machine Learning

📖 Project Overview

This project focuses on building a Sentiment Analysis model using a complete Natural Language Processing (NLP) pipeline and Machine Learning algorithms.

The goal is to classify text (reviews) into positive or negative sentiment by converting raw text into numerical features and training a model.

🚀 Features

Text preprocessing using NLP techniques

Tokenization and stopword removal

Lemmatization for better word representation

Feature extraction using TF-IDF

Model training using Logistic Regression

Model evaluation with accuracy and classification report

🛠️ Technologies Used

Python 🐍

Pandas

NumPy

NLTK

Scikit-learn

🔄 NLP Pipeline Steps

Text Cleaning

Lowercasing

Removing special characters

Tokenization

Splitting text into words

Stopword Removal

Removing common words (e.g., is, the, and)

Lemmatization

Converting words to base form

Vectorization

Using TF-IDF to convert text into numerical form

Model Training

Logistic Regression model

Evaluation

Accuracy score and classification report
